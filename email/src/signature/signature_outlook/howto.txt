1. Open the HTML in Word
2. Select All (Ctrl+A), Copy
3. Open Outlook, Create New Email, Paste
4. Select All (Ctrl+A), Copy
5. Open Signature Editor, Create New Signature, Paste, Save, OK, Close Signature Editor
6. Test
