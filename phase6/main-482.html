Transforming Print Workflows With RICOH Smart Integration

As a leading building contractor in Norway, Hent depends on printed documents for their projects. However, setting up traditional printing infrastructures at each onsite office was costly and time-consuming.

In this case study, learn how Hent utilised RICOH Smart Integration (RSI) Control+ and Ricoh multifunction printers (MFPs) to create a cloud-based print management model that simplified their print strategy and set them up for success in future projects.

During the evaluation process, we focused on cloud solutions with an intuitive interface and seamless digital workflows, plus sophisticated print management capabilities that would reduce the administrative load. […] RSI Control+ was by far the best option because of its rich functionality and user-friendliness.”
Eirik Saasen Hole, IT Consultant, Hent

Overview

Hent develops a wide range of projects, including residential developments, healthcare facilities, and maritime infrastructure across Norway, Sweden, Denmark, and Finland. They employ approximately 1,200 employees and operate from eight branch offices in locations such as Oslo, Bergen, and Ålesund.

Challenges

Moving away from an outdated and complex printing setup

Establishing essential print infrastructure at onsite offices was a labour-intensive and expensive process to secure and track print workflows—from constant MFP movement to physical server implementation and standalone software installation. This inconvenient model also led to frequent connectivity issues from the onsite servers, causing unnecessary delays.

Solutions

How RSI streamlined Hent’s print infrastructure

After evaluation, Hent selected RSI Control+ as their cloud print management solution. This enabled them to eliminate onsite servers, ease user print accessibility, and simplify the overall print processes. It also helped empower administrators by allowing them to manage user accounts and monitor printer status through web browsers.

Outcomes

Creating an agile print solution

Hent utilised RSI Control+ to create an efficient serverless print-on-demand strategy that made setting up print operations quicker and easier. The cloud model allowed for better user connectivity and assisted managers by tracking usage data for better reporting.

RSI Connect+ features and benefits

Cloud-based infrastructure: The easy-to-deploy and agile print environment eliminated the need for onsite servers and local networks.

Improved user-friendliness: Users can use their mobile devices to send print jobs and access the relevant MFPs.

Optimised project management: Managers can retrieve comprehensive usage and consumable data records, which enable accurate cost reporting for every project.

Conclusion

Printing for the future with RSI Control+

Hent used RSI Control+ to streamline their print strategy—transforming their essential printing tasks into a simple, secure, and user-friendly process.

RSI Control+ solution sets us up very well for the future and will help our users create critical documentation as we launch more construction projects across Norway and beyond. We are now exploring the potential of moving to a full print-as-a-service model, and we know that Ricoh will be on hand to help once again.”
Eirik Saasen Hole, IT Consultant, Hent
